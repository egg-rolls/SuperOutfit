# 风格体系指南

来源：央美学长帮你挑衣服（BV1Ue4y1e7SF）

## 偏日常风格

### 休闲风
- 特点：最基础、最舒适、款式最多、价格最低
- 缺点：最烂大街
- 避免烂大街：搜索"日式复古""国潮复古""港风复古"替代"休闲风"
- 搜索关键词：休闲风、日式复古、国潮复古、港风复古

### 工装风（阿美卡机）
- 特点：口袋多、面料硬挺（牛仔/拒水纤维）、超级耐造
- 适合身材：窄肩/溜肩/身材瘦（硬面料撑开肩膀，改善头肩比）
- 缺点：洗完不甩干会褶皱严重
- 名称来源：二战后传入日本，日语"美式休闲"读作 Amekaji，音译阿美卡机
- 搜索关键词（男装）：阿美卡机
- 搜索关键词（女装）：植物染、职场通勤

### 机能风
- 特点：面料硬但几乎不褶皱、遮挡范围大（下巴到裆部）、遮肉效果强
- 适合身材：身材丰腴/微胖
- 搜索关键词（男装）：山西露营
- 搜索关键词（女装）：露营风

### 学院风
- 特点：亲和力和元气值的天花板、自带校园偶像剧男主滤镜
- 价格：均价 100 左右，西装也不贵
- 适合身材：所有身材（设计来源于校服，怎么穿都不出错）
- 适合人群：高中生、学生党
- 注意：保持头发干爽、面容干净（素颜感更有魅力）
- 搜索关键词（男装）：校园休闲、DK制服
- 搜索关键词（女装）：JK宝藏、日式学妹

### 民族风
- 特点：自然、不过度修饰、异域风情、独特性最强
- 版型：宽松舒适，适合所有身材
- 适合场景：节假日出游、拍照
- 价格：比其他风格略贵（非主流，造价高）
- 搜索关键词：民族风、藏袍

## 偏正式风格

### 暗黑风
- 特点：全黑穿搭、冷静深沉、自带疏离感
- 设计重心：因为失去颜色，全部注意力在设计和材质上
- 气质：像艺术家/设计师/魔法师
- 价格：几十到几千都有，百元级也能穿出效果
- 搭配技巧：纯黑 + 闪闪发光的首饰/亮眼腰包腰带 = 画龙点睛
- 搜索关键词：暗黑风、山本耀司、黑白穿搭

### 高阶风
- 特点：源于日常但高于日常，所有服装的高级传达版
- 例子：西装 → 换低领背心 + 短裙裤 + 皮靴 = 高阶风
- 缺点：价格太贵
- 搜索关键词：设计师

## 淘宝搜索技巧

- 店铺榜 → 按风格分类 → 按质量/价格/服务/评论排名
- 找到自己的风格 = 找到自己的衣柜
- 养成习惯：看见喜欢的衣服就点收藏
- 淘宝算法会学习你的偏好，搜索结果会越来越精准

## 风格与身材速查

| 风格 | 适合身材 | 核心优势 |
|------|----------|----------|
| 休闲 | 所有 | 舒适、百搭 |
| 工装 | 窄肩/溜肩/瘦 | 硬面料撑肩、改善头肩比 |
| 机能 | 丰腴/微胖 | 大面积遮肉 |
| 学院 | 所有 | 校服设计，不出错 |
| 民族 | 所有（宽松版型） | 遮盖瘦削或丰腴 |
| 暗黑 | 所有 | 全黑显瘦、设计感强 |
| 高阶 | 身材好加分 | 设计感、正式感 |
