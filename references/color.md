# 色彩搭配规则

来源：穿搭入门终极指南

## 万能色（无脑买，互相随便搭）

- 黑、灰、白、藏青色系、大地色系（卡其/驼/棕）
- 这几类颜色互相之间可以随意组合
- 它们跟其他颜色也能协调搭配

## 四种颜色搭配模型

### 1. 同色系（永远不出错）
- 全黑：靠材质差异打破单调（棉+皮+金属）
- 全白、全灰、全大地色同理
- 深浅搭配：深灰+浅灰，深棕+浅卡其
- 同色系混搭零风险

### 2. 点缀色（最简单的出彩方式）
- 全身同色系 + 1 个反差色单品
- 点缀色面积不超过全身 10%
- 例：一身黑 + 红色帽子
- 例：一身白 + 黑色腰带
- 只需要一个点缀色就够

### 3. 颜色呼应（穿搭感的来源）
- 非相邻单品之间颜色统一
- 鞋子和上衣同色
- 裤子和帽子同色
- 配件之间的颜色协同
- 呼应通常通过配件实现

### 4. 撞色（进阶玩法，普通人慎用）
- 五颜六色混搭
- 需要很强的色彩感知力
- 日常通勤不建议

## 选一个属于自己的颜色

- 除了万能色之外，选 1 个自己喜欢的颜色
- 所有非基础单品尽量往这个颜色靠
- 例：选红色 → 红色包、红色丝巾、红色帽子
- 需要点缀时随手抓起几个同色系配件就能呼应
- 不要乱买各种颜色 → 根本无法协同

## 肤色与颜色

- 黑黄皮：避开冷色系（蓝/绿/紫）
- 黑黄皮：避开高饱和度颜色
- 黑黄皮：低饱和度冷色系可以驾驭
- 白皮：什么颜色都能驾驭

## 饰品颜色体系

- 饰品分两大色系：金色系、银色系
- 尽量在同一色系内购买
- 金色+银色混搭会产生奇怪效果
- 选定一个色系就一直买下去
