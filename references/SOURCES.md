## #6 色彩模型工程化

- **标题：** 色彩评分数学模型构建（内部工程）
- **作者：** SuperOutfit 开发团队
- **平台：** 内部
- **链接：** N/A
- **处理日期：** 2026-06-02
- **内容摘要：** 从 5 个来源爬取 6400 组真实色卡，提取 42 维特征（27 HSL + 15 OKLab），训练高斯过程模型（MAE 2.66，比线性回归提升 21%）
- **拆解产出：**

| 文件 | 内容 |
|------|------|
| `scripts/scrape_palettes.py` | Color Hunt + Colormind + LOL Colors + Design Seeds 色卡爬虫 |
| `scripts/scrape_wxsecai.py` | 色采 (wxsecai.com) 色卡爬虫 |
| `scripts/train_color_model.py` | 特征提取 + 线性回归 + GP 训练 |
| `scripts/color_math.py` | 模型预测接口（替代旧的色卡直接匹配） |
| `data/raw_palettes.json` | 6400 组原始色卡数据 |
| `data/color_model.pkl` | 训练好的模型文件 |
| `references/color_pipeline.md` | 完整数据管道文档 |

- **待升级：** XGBoost/GP 非线性模型、OKLab 色彩空间特征、更多数据源
