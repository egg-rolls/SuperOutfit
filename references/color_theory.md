# 色彩理论与配色数学

来源：小关 配色小管家（BV1YLPozVEhR 系列）

## 核心原理：世界名画色卡

4万张世界名画色卡的统计分析发现：所有好看的配色都遵循可量化的数学规律。

### CMYK 四分色模型
- 任何颜色都可以用青(C)、品红(M)、黄(Y)、黑(K)的占比来表示
- 每组配色中，CMYK 某一项或两项占比偏高 → 控制整体色调统一

### 底色理论
- 绘画前先铺一层底色（蓝/黄/红），创造画面颜色的整体基调
- 此后每一笔都会晕染上底色 → 整体颜色统一
- 穿搭同理：确定一个底色基调，所有配色在底色安全范围内

### 底色的应用
1. 从全身主色调中提取最主导的颜色倾向（暖/冷/中性）
2. 暖底色 → 暖色系内自由搭配
3. 冷底色 → 冷色系内自由搭配
4. 所有颜色都"晕染"上底色调性 → 视觉和谐

## 色环配色关系（HSL 色彩空间）

### 色相环（Hue，0-360°）
```
    0° 红
  330°   30°
300°       60° 黄
  270°   90°
   240° 蓝  120° 绿
  210°   150°
    180° 青
```

### 五种配色关系

| 关系 | 色相差 | 效果 | 适合场景 |
|------|--------|------|----------|
| 同色系 | 0° | 统一、安全、不出错 | 日常、通勤 |
| 类似色 | 15-30° | 和谐、柔和、有层次 | 日常、约会 |
| 互补色 | 180° | 对比强烈、视觉冲击 | 拍照、街头 |
| 分裂互补 | 150° 或 210° | 对比但不冲突 | 进阶搭配 |
| 三角配色 | 120° | 平衡、有活力 | 风格化穿搭 |

### 明度（Lightness）
- 高明度（浅色）+ 低明度（深色）= 有层次
- 全高明度 = 轻盈、柔和
- 全低明度 = 沉稳、深沉
- 明度差太小 → 模糊、没层次

### 饱和度（Saturation）
- 高饱和 = 鲜艳、抢眼
- 低饱和 = 高级、柔和
- 全身高饱和 → 只适合拍照，不适合日常
- 日常建议：大面积低饱和 + 小面积高饱和点缀

## 穿搭配色的安全规则

### 底色安全区
- 确定底色后，所有配色在底色的色相 ±30° 范围内 = 安全区
- 安全区内任意搭配都和谐

### 面积比规则
- 主色（大色块）：60-70%
- 辅助色（中色块）：20-30%
- 点缀色（小色块）：5-10%
- 点缀色可以跳出底色安全区

### 无彩色（黑白灰）规则
- 黑白灰不含色相，可以和任何颜色搭配
- 无彩色不计入"颜色数量"
- 全身无彩色 + 1个有彩色点缀 = 最安全的出彩方式
