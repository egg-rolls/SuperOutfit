# 叠穿层次指南

来源：造型师 土气？头大？不伦不类？6个技巧轻松叠穿（BV1Ub4y127dd）

## 三角区理论

叠穿最关键的位置是**三角区**：从领口 → 胸部 → 腰腹 → 下摆的区域。所有内搭分层、领口清晰的结构都在这个位置。

## 三个叠穿公式（秋冬）

### 公式 1：高领打底 + V领针织 + 外套
- 最内层：薄高领/半高领
- 中间层：V领针织或V领毛衣
- 外套层：大衣/夹克

### 公式 2：T恤/高领 + 帽衫 + 外套
- 最内层：T恤或薄高领
- 中间层：帽衫（连帽卫衣）
- 外套层：外套

### 公式 3：圆领T恤 + 衬衫 + 西装
- 最内层：圆领T恤
- 中间层：衬衫（露出一道边）
- 外套层：西装/西服夹克

## 核心规则：外包里

- 最外面的外套必须兜住里面所有层次
- 不要里长外短太夸张 → 显得比例失调

## 领口层次规则

- 里/中/外三层最好用不同领型叠加 → 更有叠穿感、更精致
- ❌ 两个卫衣帽子叠一起 → 经典叠穿翻车，显邋遢
- 保持领口每一件的边缘界线清晰
- ❌ 厚高领（又厚又囊）→ 破坏领口秩序感，显臃肿、没脖子
- ✅ 薄半高领作为最内层 → OK
- 短脖子/下颌线不清晰 → 避免厚高领
- 丝巾、围巾、项链 → 丰富领口层次的重要单品

## 腰身与下摆规则

- 内搭每一层露出来的面积越平均 → 越不显臃肿、越显瘦
- 原理：色块切割，不会让肚子区域变成一整片
- 需要扎裤子时：内搭要比外套更紧更薄（内紧外松、内薄外厚）
- Cityboy 风格：不扎裤子，利用里外长短/版型不一致突出叠穿感
- 下摆的这道"劈叉"就是叠穿的层次感

## 叠穿配色规则

### 秩序感是核心
- 颜色必须有主色
- 叠穿是多单品排列组合，没有主色 → 不协调

### 纯色为主
- 印花/纹样/亮色 → 只做小面积点缀色
- 亮色当第二层内搭，露一个小边边当过渡色
- 大色块不超过 3 个颜色

### ABCDE 字母配色法
- **ABA**：最里层和最外层同色，中间不同
- **AAB**：内搭全同色，外套不同色
- 黑白灰无彩色叠穿 → 不费脑子也好看

### 呼应色
- 配饰（鞋/包/帽子）和身上衣服同色或同色系

## 不靠叠加衣服数量的叠穿（适合南方/沿海）

### 两件式外包里叠穿
- T恤外面套坎肩/背心
- 长袖外面套工装背心/针织无袖
- 不需要外套也能有叠穿效果

### 用配饰做层次（不靠衣服件数）
- 项链压着领子
- 双肩包/斜挎包的带子
- 袜子对腿部的遮挡
- 帽子
- 手套
- 这些位置都带上东西 → 造型完成度高 → 层次丰富
- 层次的对立面是"单薄无遮挡"，不是衣服件数

### 单件衣服自身的层次
- 夹两件设计（拼接元素）
- 扭曲/褶皱/镂空设计
- 这种衣服本身层次丰富，适合单穿，不适合再叠加

### 穿着方式做层次
- 衬衫解开一颗扣子，人为制造一道折
- 法式塞（单边塞）→ 上下之间有遮挡关系 = 有里外之分 = 有层次
- 最简单的叠穿方式

### 反季节搭配（南方适用）
- 上半身长袖（露出面积少）+ 下半身短裤（露出面积多）
- 用衣服色块切割露肤度 = 层次感
- 露出的皮肤和衣服之间的遮挡关系本身就是一层层次
