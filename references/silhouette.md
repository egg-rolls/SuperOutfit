# 版型与身材匹配

来源：穿搭入门终极指南

## 上衣四种版型

| 版型 | 特点 | 适合身材 |
|------|------|----------|
| 阔型 | 宽松、遮肉 | 上身缺陷多：过胖、肚子大、肩膀难看 |
| 短宽/倒梯型 | 短款、宽肩 | 五五分身材：上身过长、腿过短（拉高腰线） |
| 正型 | 合身、不出错 | 普通身材，无明显缺陷 |
| 修身 | 贴身、凸显线条 | 身材好：肌肉明显、想展示身形 |

- 身材正常/良好的人可以兼顾阔型和短宽，看是否追求风格化
- 通过衣服的平铺图可以判断版型

## 裤子四种版型

| 版型 | 特点 | 适合身材 |
|------|------|----------|
| 阔型 | 宽松、遮盖力强 | 下半身缺陷多：腿短/粗/弯/O型腿 |
| 锥型 | 大腿宽松、小腿收窄 | 腿型难看+腿短：遮丑同时显利落 |
| 正型 | 标准、不出错 | 普通身材 |
| 修身 | 贴身、凸显腿部 | 腿细、想展示腿部曲线 |

## 身材问题与解决方案

### 头大
- 用耳钉把视觉重心下移
- 选择 55-65cm 长项链，把视觉比例往下拉
- 靴子拉高身形，弱化头大
- 冷帽/渔夫帽遮到耳朵以下，修饰脸型

### 腿短
- 选择短宽/倒梯型上衣拉高腰线
- 腰带固定腰线位置
- 靴子增加下半身视觉比例
- 避免过长的上衣

### 五五分身材
- 短款上衣 + 高腰裤
- 腰带是核心配件
- 靴子增加下半身比例
- 长项链把视觉重心下移

### 脖子粗
- 选择 55-65cm 主链（垂到胸口）
- 避免 45-55cm 短项链
- 视觉重心下移

### 脖子细
- 可以戴 choker 让脖子看起来粗壮一点
- choker + 主链混搭

## 帽子与脸型

| 帽型 | 适合 |
|------|------|
| 弯沿棒球帽 | 脸尖、脸型好看、想搞氛围感 |
| 平沿棒球帽 | 脸宽/大/肥/圆/脖子粗 |
| 渔夫帽 | 绝大部分脸型，遮耳朵以下修饰颧骨 |
| 冷帽 | 脸大/圆/头大，遮到耳朵以下有修脸效果 |
| 发带 | 额头高、发际线高 |
