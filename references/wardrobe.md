# 衣橱结构与日常穿搭流程

来源：穿搭入门终极指南

## 衣橱五大结构

所有穿搭都可以拆解为五层：

```
打底 → 裤子 → 外套 → 配件 → 鞋
```

每天按这个顺序选择，就是穿搭的完整流程。

## 第一步：选择打底

- 起床第一件事：看气温选打底
- 热了穿背心/T恤，冷了穿卫衣/毛衣
- 基础配置：黑色 + 白色两个颜色的打底
- 常见类型：背心、短袖T恤、长袖T恤、卫衣、衬衫、Polo衫、毛衣
- 打底和外套的界限模糊：衬衫/卫衣既可做打底也可做外套
- 建议：床边安排衣架挂打底，随时保持平顺可穿

## 第二步：选择裤子（核心）

**裤子是男生穿搭的内核，决定了整体风格走向。**

四种裤子类型：

| 类型 | 风格 | 场景 |
|------|------|------|
| 卫裤/运动裤 | 舒适、运动 | 居家、轻运动 |
| 工装裤 | 硬挺、多细节、强风格 | 复杂搭配、工装风 |
| 牛仔裤 | 耐穿耐脏、百搭 | 日常、几乎所有场景 |
| 西裤/休闲裤 | 垂顺、正式感 | 上班、偏正式场合 |

- 裤子一旦定了，穿搭风格就锁定了
- 建议：床边收纳裤架，随时就绪

## 第三步：选择外套（由裤子决定）

- 外套是风格化最丰富、体现穿搭风格最强烈的单品
- 外套由裤子的风格和颜色决定
- 卫裤配卫衣，西裤配西服夹克
- 裤子是什么风格，外套就跟着走
- 常见外套：衬衫外套、工装夹克、牛仔夹克、飞行员夹克、西服、冲锋衣、帽衫、棉服

## 第四步：选择配件

- 打底和裤子成型后，开始搭配配件
- 饰品、帽子、腰带、丝巾、包
- 通过颜色呼应来协同配件和整体穿搭

## 第五步：选择鞋子

- 裤子的风格决定鞋子的风格
- 上身的颜色决定鞋子的配色
- 颜色呼应是关键

## 上线穿搭 vs 下线穿搭

### 下线穿搭（80% 的日子）
- 不需要思考，无脑穿
- 快速套上，大概率不出错
- 有基本造型感但不夸张
- 适用：上学、加班、普通聚会
- 衣橱 80% 应该是这种衣服

### 上线穿搭（偶尔重要时刻）
- 有搭配门槛，需要花时间搭配
- 更贵、更有排面、更风格化
- 不会每天都穿
- 适用：约会、重要社交场合
- 少量配置，不要买太多

## 基础搭配逻辑

- 一身素色 → 加一个有 logo 的单品
- 身上 logo → 控制在 1-2 件
- 穿搭太单调 → 丝巾是最万能的丰富手段
